## 1. 자료형 (Data Types)

### 1.1 숫자형 (Number)

JavaScript는 정수와 소수를 모두 `number` 타입으로 처리한다.

```javascript
let a = 10;     // number
let b = 3.14;   // number

console.log(a + b);     // 13.14

console.log(typeof a);
console.log(typeof b);
```

💡 **포인트**: Python과 달리 `int`, `float` 구분 없이 모든 숫자는 `number` 타입!

---

### 1.2 문자열 (String)

작은따옴표(`'`)나 큰따옴표(`"`)로 문자열을 선언한다.

```javascript
let name = 'Alex';
let message = 'Hello';

console.log(name);
console.log(message);

console.log(typeof name);
console.log(typeof message);
```

---

### 1.3 불리언 (Boolean)

참(`true`) 또는 거짓(`false`) 값을 가지는 타입.

```javascript
let isAdult = true;
let hasTicket = false;

console.log(isAdult);
console.log(typeof hasTicket);
```

---

### 1.4 null과 undefined

#### undefined - 값이 할당되지 않은 상태

```javascript
let score;
let a = 10; 
let b = 3;  

console.log(score);         // undefined
console.log(typeof score);  // undefined

console.log(Math.floor(a/b));  // 3 (버림)
console.log(10 % 3);           // 1 (나머지)
```

#### null - 의도적으로 "비어있음"을 표현

```javascript
let selectedUser = null;

console.log(selectedUser);       // null
console.log(typeof selectedUser); // object (⚠️ JavaScript의 유명한 버그!)
```

💡 **undefined vs null**
- `undefined`: 변수를 선언만 하고 값을 안 넣은 상태
- `null`: 개발자가 "여긴 지금 비어있어"라고 명시적으로 표현한 것

## 2. 연산자 (Operators)

### 2.1 비교 연산자

크기를 비교하여 `true` 또는 `false`를 반환한다.

```javascript
console.log(10 > 5);   // true
console.log(10 < 5);   // false

console.log(10 >= 10); // true
console.log(10 <= 5);  // false
```

---

### 2.2 동등 연산자

`==`와 `===`의 차이를 이해하는 것이 중요!

```javascript
console.log(10 == "10");   // true (값만 비교, 타입 변환 발생)
console.log(10 === '10');  // false (값과 타입 모두 비교)
```

⚠️ **주의**: 실무에서는 **항상 `===` (엄격한 동등) 사용을 권장**한다!

---

### 2.3 논리 연산자

여러 조건을 조합할 때 사용한다.

```javascript
let age = 20;
let hasId = true;

console.log(age >= 19 && hasId);  // true (AND: 둘 다 참이어야)
console.log(age < 19 || hasId);   // true (OR: 하나만 참이어도)
console.log(!hasId);              // false (NOT: 반대로)
```

| 연산자 | 의미 | 설명 |
|:---:|:---:|:---|
| `&&` | AND | 모든 조건이 참일 때 참 |
| `\|\|` | OR | 하나라도 참이면 참 |
| `!` | NOT | 참↔거짓 반전 |

---

### 2.4 문자열 연결 연산자

문자열을 합치는 여러 가지 방법.

```javascript
// 문자열 연결(Concatenation) 연산자

let firstName = 'Alex';
let lastName = 'Kim';

console.log(firstName + " " + lastName);  // Alex Kim
console.log('Mr.' + lastName);            // Mr.Kim

console.log(`Hello, ${firstName}`);       // Hello, Alex (템플릿 리터럴)
```

💡 **템플릿 리터럴 팁**
- 백틱(`` ` ``)은 ESC 키 아래에 있음
- 맥에서는 `Option(⌥)` + `` ` `` 로 입력

## 3. 조건문 (Conditionals)

### 3.1 계산기 만들기 실습

`if/else if/else` 구조를 활용한 간단한 계산기.

```javascript
// 실습: 계산기 만들기
// 1. 숫자 두 개 변수 할당
let a = 10;
let b = 5;

// 2. 산술연산자 변수 할당
let operator = "+";     // +, -, *, /

// 3. 선택한 연산자에 따라서 계산해서 결과를 출력하는 프로그램 완성
// Hint: 조건문(if/else) 사용

if (operator === "+") {
    console.log(a + b);
} else if ( operator === "-") {
    console.log(a - b);
} else if ( operator === "*") {
    console.log(a * b);
} else if ( operator === "/") {
    console.log(a / b);
} else {
    console.log('지원하지 않는 연산자입니다')
}
```

---

### 3.2 합격 판정기 실습

복잡한 조건을 논리 연산자로 조합하는 예제.

```javascript
// 실습: 합격 판정기

// 점수(score) 70 이상
// 출석률(attendance) 80 이상
// 과제 제출 여부(submitted) true
// 단, 점수가 90점 이상이면, 출석률 무시하고 합격

// 출력: "합격" 또는 "불합격"

let score = 80;
let attendance = 90;
let submitted = true;

if (
    submitted && (
        score >= 70 && attendance >= 80 || (score > 90)
    )
) {
    console.log("합격");
} else {
    console.log('불합격');
}
```

💡 **합격 조건 정리**
1. 과제 제출 필수 (`submitted === true`)
2. 그리고 아래 중 하나를 만족:
   - 점수 70점 이상 **AND** 출석률 80% 이상
   - 또는 점수 90점 초과 (출석률 무시)

## 4. 반복문 (Loops)

### 4.1 for문

정해진 횟수만큼 반복할 때 사용.

```javascript
// 몇 번 반복할건지
// python으로 15번 반복

// for (초기값; 조건식; 증가식) {
//     console.log("hello");
// }

for (let i = 0; i < 3; i++) {
    console.log("hello");
}
```

📝 **for문 구조**
```
for (초기값; 조건식; 증가식) {
    // 반복할 코드
}
```

---

### 4.2 while문

조건이 참인 동안 계속 반복.

```javascript
let i = 0;

while (i < 3) {
    console.log('${i} hello');
    i++;  // 이 증가식이 있어야 위 조건식이 의미가 있다
          // 예> 이게 없으면 다음 i도 0이 되어 2까지란 조건이 있든말든 
          //     계속 0만 반복되는 무한루프
}
```

⚠️ **무한루프 주의!**: `i++`를 빼먹으면 영원히 루프가 돈다!

## 5. 함수 (Functions)

### 5.1 기본 함수 정의

```javascript
function add(a, b) {
    return a + b;
}

let result = add(3, 5);
console.log(result);  // 8
```

---

### 5.2 평균 구하기 함수

배열의 평균을 구하는 함수 두 가지 버전.

#### 버전 1: 기본 for문 사용

```javascript
function getAverage(scores) {
    if (scores.length === 0) return 0;
    let sum = 0;
    for (let i = 0; i < scores.length; i++) {
        sum += scores[i];
    }
    return sum / scores.length;
}

console.log(getAverage([80, 90, 70]));  // 80
```

#### 버전 2: for...of 사용

```javascript
// of 이용한 방법
function getAverage(scores) {
    if (scores.length === 0) {
        throw new Error()
    }
    let sum = 0; 
    for (const score of scores) {
        sum += score;
    }
    return sum / scores.length;
}

let result = getAverage([80, 85, 92, 97]);
console.log(result);  // 88.5
```

## 6. 배열과 객체 (Array & Object)

### 배열 (Array)

다양한 타입의 요소를 담을 수 있는 리스트.

```javascript
let numbers = [10, 20, 30, 'one', 'two'];

// 인덱스로 접근
// console.log(numbers[0]);  // 10
// console.log(numbers[3]);  // 'one'

// 방법 1: 기본 for문으로 순회
// for (let i = 0; i < numbers.length; i++) {
//     console.log(numbers[i]);  // 전체 순회
// }

// 방법 2: for...of (제일 많이 씀!)
for (const num of numbers) {
    console.log(num);
}

// break 사용 예시
// for (const num of numbers) {
//     if (num === 30) {
//         break
//     }
//     console.log(num);
// }

// 방법 3: forEach (break 못 넣는 단점 있음)
// numbers.forEach(score => {
//     console.log(score);
// })

// Python enumerate처럼 인덱스와 값 같이 가져오기
// for (const [i, num] of numbers.entries()) {
//     console.log(i, num);
// }
```

### 객체 (Object)

키-값 쌍으로 데이터를 구조화.

```javascript
let user = {
    name: 'alex',
    age: 20,
};

console.log(user.name);     // 'alex' (점 표기법)
console.log(user['age']);   // 20 (괄호 표기법)
```

💡 **배열 순회 방법 비교**

| 방법 | break 가능 | 인덱스 접근 | 추천도 |
|:---|:---:|:---:|:---:|
| `for (let i=0; ...)` | ✅ | ✅ | ⭐⭐ |
| `for...of` | ✅ | ❌ (별도 처리) | ⭐⭐⭐ |
| `forEach` | ❌ | ✅ (콜백 인자) | ⭐⭐ |
| `for...of + entries()` | ✅ | ✅ | ⭐⭐⭐ |
